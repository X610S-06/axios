# Axios

This demonstrates the functionalities of axios API using JavaScript.

# Clone and Use 

  Open Terminal 
  
    git clone 'https - copied url'

  Change directory  

    cd 'directory name'

  Open all files on VS code  

    code .
    
  Install dependencies

    npm install
    
  Open index.html with Live server
